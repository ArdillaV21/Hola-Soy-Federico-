Experiencia

Hace dos años me encuentro aprendiendo diferentes lenguajes y herramientas en el mundo desarrollo web. Entre ellos se encuentran los siguientes.

Lenguajes

  • HTML Lenguaje de marcado utilizado para crear y estructurar páginas web.
  • CSS Lenguaje utilizado para describir la presentación de documentos HTML, como colores, diseño y fuentes.
  • JS Lenguaje de programación que permite la creación de contenido dinámico e interactivo en páginas web.
  • PHP Lenguaje de programación ampliamente utilizado para el desarrollo web del lado del servidor.
  • C# Lenguaje de programación desarrollado por Microsoft, utilizado principalmente en aplicaciones de escritorio y web.

Herramientas

  • Bootstrap Framework de desarrollo frontend que facilita la creación de diseños web responsivos y atractivos.
  • LESS Lenguaje de hojas de estilo dinámico que amplía CSS con características como variables y funciones.
  • SASS Preprocesador CSS que ofrece características como variables, anidación y mixins para mejorar la eficiencia del desarrollo.
  • NodeJS Entorno de ejecución de JavaScript del lado del servidor, utilizado para construir aplicaciones web escalables.
  • ReactJS Biblioteca de JavaScript desarrollada por Facebook para crear interfaces de usuario interactivas y rápidas.
  • Vue Framework progresivo de JavaScript utilizado para construir interfaces de usuario interactivas.
  • MongoDB Base de datos NoSQL que utiliza documentos JSON para almacenar datos, popular por su flexibilidad y escalabilidad.
